# 🏆 Проект: Турнирная таблица (TournamentTable)

## 📌 Описание проекта
Данное приложение предназначено для ведения и управления турнирными таблицами спортивных команд по различным видам спорта (Футбол, Баскетбол, Волейбол). Программа разделена на логические слои, поддерживает сохранение данных в форматы JSON/XML, фильтрацию сезонов, дисквалификацию участников и расширенную систему сортировки мест.

---

## 🏗 Архитектура проекта

Проект строго разделен на два основных модуля:
1. **TournamentTable (Основной проект)** — графический интерфейс (WPF/XAML), формы выбора сезонов, отображение таблиц и диалоговые окна.
2. **Model (Библиотека классов)** — изолированная бизнес-логика игры. Внутри разделена на папки:
   - **Core** — базовые классы команд, логика подсчета очков, интерфейсы и механика турнира.
   - **Data** — сериализация и десериализация файлов.

### 🔹 Базовая часть (5 баллов)
* [ ] **Базовый класс Team:** Создан класс `Team` со свойствами для названия, побед, ничьих, поражений и абстрактным/виртуальным свойством `CalculatePoints`.
* [ ] **Специфичные классы команд:** Реализованы классы-наследники `FootballTeam`, `BasketballTeam` и `VolleyballTeam`, каждый из которых по-своему переопределяет начисление очков (`CalculatePoints`) в соответствии с правилами вида спорта.
* [ ] **Интерфейс ITable:** Определяет методы для добавления результатов матчей `Match(Team, Team)` и методы сортировки `SortDefault()` и `SortByScore()`.
* [ ] **Класс TournamentTable:** Хранит название, год сезона, массивы команд и сыгранных матчей, а также реализует интерфейс `ITable`.
* [ ] **Тестовое наполнение данных:** Программно генерируется от 10 до 20 различных команд и не менее 3 полноценных турниров в разных сезонах без одновременного пересечения команд в один момент.
* [x] **Главное меню:** Реализованы выпадающие списки для выбора вида спорта и года сезона, а также кнопка «Показать таблицу», активирующаяся при выборе обоих параметров.
* [x] **Окно турнирной таблицы:** Таблица открывается в новом окне, изначально отсортированная по алфавиту (по имени команд). Данные автоматически считываются из JSON-файлов (создаваемых при первом запуске).
* [x] **Кнопки сортировки:** Рядом с таблицей размещены кнопки «Отсортировать по имени» и «Отсортировать по очкам».
